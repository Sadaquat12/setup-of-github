How to setup of Github

1. create a folder on github
2. go to on
a. git clone (paste the url of github folder)
b. press cd and press tab for your required github folder
c. write .. "code ." and press enter (for VS Code)
d. work some on your folder and ctrl + S

How to push on github

same as write on cmd
1. git add .
2. git commit -m "first"
3. git push
